1.1.4 - Изменены названия кнопок с анг. на русский в окне таблицы ВКС, а также в главном меню.
1.1.5 - Установленны иконки для каждой страницы приложения.
1.1.6 - Добавлен функционал удаления событий ВКС по диапозону дат, в интерфейсе настроек.
1.1.7 - Визуальные изменения, убраны записи в таблице ВКС из файла tableListVKS.html , таблица отрисовывается только динамически (посредством javascript).
1.1.8 - Визуальные изменения, убрана отрисовка кнопки "Печать" в таблице ВКС ( на период пока она не функционирует)

1.2.0 - 20.01.2026 / Реализованна выгрузка в Word файл списка событий по заданому диапазону дат
1.2.1 - 21.01.2026 / Реализованно заполнение строки диапазона дат, в файле выгрузки Word
1.2.2 - 21.01.2026 / При выгрузке в Word файл ВКС событий, имя файла генерируется автомотически на основе текущей даты.
